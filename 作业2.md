### 最左推导
```
<赋值>                 => <标识符> = <表达式>

<标识符> = <表达式>     => B = <表达式>

B = <表达式>           => B = <标识符> * <表达式>

B = <标识符> * <表达式> => B = C * <表达式>

B = C * <表达式>       => B = C * (<表达式>)

B = C * (<表达式>)     => B = C * (<标识符> * <表达式>)

B = C * (<标识符> * <表达式>) => B = C * (A* <表达式>)

B = C * (A* <表达式>)  => B = C * (A* <标识符> + <表达式>)


B = C * (A* <标识符> + <表达式>) => B = C * (A* C + <表达式>)

B = C * (A* C + <表达式>) => B = C * (A* C + <标识符>)

B = C * (A* C + <标识符>) => B = C * (A* C + B)

```
